# Mantenimiento-TGM-Analisis
Análisis de datos sobre el mantenimiento de equipos de seguridad en Talleres Gráficos de México (2025). El proyecto enfoca el rol del Data Analyst en la continuidad operativa, identificando riesgos por falta de registros y optimizando la planeación de activos críticos.
 Descripción del Proyecto
Este repositorio contiene un análisis de datos enfocado en la gestión de mantenimiento de los equipos de seguridad e infraestructura de Talleres Gráficos de México para el primer semestre de 2025.
El objetivo es demostrar cómo un Data Analyst aporta valor al identificar riesgos operativos y proponer optimizaciones basadas en datos reales.
 Resumen de los Datos
El dataset incluye registros de diversos servicios esenciales para la planta, tales como:
Sistemas de Seguridad: Circuito Cerrado de TV (CCTV), Rayos X y Arco detector de metales.
Infraestructura Crítica: Sistemas hidroneumáticos, equipos contra incendio y elevadores de carga.
Servicios de Planta: Aire acondicionado, luminarias LED y mantenimiento de áreas verdes.
 Hallazgos Clave (Insights de Datos)
Tras un análisis preliminar de los datos:
Recurrencia Operativa: El sistema de CCTV y los servicios de jardinería muestran una continuidad mensual clara (de enero a julio), lo que facilita la planeación presupuestaria.
Identificación de Riesgos: Activos críticos como el sistema contra incendio y los elevadores de carga aparecen "sin dato" en la columna de mes, 
lo que representa una oportunidad para mejorar la gobernanza de datos y evitar fallas imprevistas.
Especialización: Los servicios de seguridad de alta tecnología (Rayos X) están programados específicamente para el segundo trimestre (mayo y junio).
 Herramientas Utilizadas
Python: Limpieza de datos y manejo de nulos (Pandas).
Matplotlib/Seaborn: Visualización de la carga de trabajo mensual.
